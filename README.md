# MotorPH Payroll Processing System
A Java with Maven-based system that manages MotorPH employees' payrolls.

This system automates payroll processing by calculating net and gross pay based on hours worked 
within a set payment period, while incorporating benefits and mandated government deductions.

# Tech Stack:
Language: Java 25.0.1

Build Tool: Maven 3.9+

Data Format: CSV

# Prerequisistes:
-JDK 21 or higher (Managed via SDKMAN! recommended)
- Maven
- Ubuntu/WSL2 (if running in a Linux environment)
- Any terminal
